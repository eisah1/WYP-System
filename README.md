# WYPSystem

A university software engineering project focused on developing a police management system designed to manage records, user interactions and operational workflows.

## Features

* User authentication and access control
* Record and data management
* Interactive user interface
* Database integration
* Google Maps API integration
* Form validation and error handling

## Technologies Used

* Java
* SQL
* HTML/CSS
* Google Maps API
* Git

## Project Overview

Developed as part of university coursework to apply software engineering principles, object oriented programming and database integration within a real world style application. The project included Google Maps integration to support location based functionality and improve system usability, while following structured development and testing practices.
